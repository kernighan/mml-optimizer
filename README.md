# example usage

On the command line:

``` sh
$ mml-optimizer c8d8e8f8g4
```

With the API:

``` js
var opt = require('mml-optimizer');
var optimizedMml = opt('c8d8e8f8g4');
```

# installation

Just plain ol' [npm](https://npmjs.org/) installation:

``` sh
npm install mml-optimizer
```
